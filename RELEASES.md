# Release Convention — VendorCompare

## Tagging
- Every build shipped to users gets a git tag: `vMAJOR.MINOR.PATCH`
- Tag the commit that was built, not a later one
- Command: `git tag -a v1.2.3 -m "Release v1.2.3" && git push origin v1.2.3`
